# Weather Intelligence System 🌦️

A real-time weather forecasting application that fetches data from the OpenWeather REST API. This project demonstrates asynchronous JavaScript and dynamic UI/UX design.

### Key Features:
* **Live API Integration:** Fetches temperature, humidity, and wind speed using `Async/Await`.
* **Dynamic Themes:** The background and greetings change automatically based on weather conditions (Rain, Clear, Haze, etc.) and time of day.
* **Wardrobe Advisor:** Custom logic that suggests appropriate clothing based on current temperature.

### Tech Stack:
* HTML5, CSS3 (Glassmorphism), JavaScript (ES6+), OpenWeather API.
